| Concept            | Parameters                                          | Algorithm                             | Python                         |
|--------------------|-----------------------------------------------------|---------------------------------------|--------------------------------|
| New Variable       | name, type, initVal                                 | Create variable with initial value    | name = initVal                 |
| Output             | message                                             | Output the message                    | print(message)                 |
| Input              | variable, message                                   | Ask user and store result             | variable = input("message")    |
| Convert to Integer | oldVariable, intVariable                            | Convert oldVariable to integer        | intVariable = int(oldVariable) |
| Branch             | condition, code                                     | If condition is true, execute code    | if condition:                  |
| For Loop           | sentry, start, finish, change                       | Repeat from start to finish by change | for sentry in range(...):      |
| While Loop         | sentry, initialization code, condition, change code | Repeat while condition is true        | while condition:               |
| Function           | functionName, return type, parameters               | Create reusable function              | def functionName(parameters):  |
