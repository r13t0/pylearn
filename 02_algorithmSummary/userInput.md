🧩 Pattern Name: User Input Greeting

📌 Description
A simple interaction pattern where the program:

- asks the user for input
- stores it in a variable
- uses it immediately in a printed response
- Used for basic “ask → respond” programs.

Used for basic “ask → respond” programs.

🔁 Pattern Template
# USER INPUT GREETING PATTERN

name = input("Enter your name: ")
print("Hello", name)

⚙️ Pattern Structure

Step 1: Prompt user with input()
Step 2: Store response in a variable
Step 3: Use variable in output

🌍 Reusable In
Any language (Python, JavaScript, Java, C++)
Any simple interaction system:
login prompts
forms
chatbots
CLI tools

💡 Key Insight
Input → Store → Reuse immediately in output
