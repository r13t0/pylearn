# Classification

## Purpose

Place data into one of several categories based on rules or ranges.

## Recognition Clues

- Which category?
- Assign a grade.
- Determine the type.
- Place into a group.

## Mental Model

Sorting letters into labeled mailboxes.

## Algorithm

Receive data

Compare against category rules

Choose one matching category

Return category

## Programming Concepts

- Conditions
- Comparison Operators
- Boolean Logic

## Python Toolbox

if
elif
else

## Sample Code

```python
age = int(input("Age: "))

if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")
```
