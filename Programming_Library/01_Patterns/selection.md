# Selection

## Purpose

Choose between different actions based on a condition.

## Recognition Clues

- If...
- Otherwise...
- Depending on...
- When this happens...

## Mental Model

A traffic light choosing a direction.

## Algorithm

Check condition

If true:
    perform action A

Else:
    perform action B

## Programming Concepts

- Boolean expressions
- Conditions
- Branching

## Python Toolbox

if
elif
else

## Sample Code

age = int(input("Age: "))

if age >= 18:
    print("Allowed")
else:
    print("Not allowed")
