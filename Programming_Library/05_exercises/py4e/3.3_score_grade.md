Excellent. This exercise introduces your **first real classification problem**.

Up until now, you've mostly made **one decision** ("yes" or "no"). Here, you're deciding **which category** a value belongs to.

# Step 1 — Problem Analysis (Decomposition)

## Purpose

Determine a letter grade from a numeric score.

## Inputs

* Score (decimal number)

Example:

```text
0.85
```

## Outputs

* A, B, C, D, F
* or an error message if the score is invalid.

## Constraints

* Score must be between 0.0 and 1.0.
* Invalid scores produce an error.
* Only one grade should be printed.

## Sub-Problems

1. Get the score.
2. Validate the score.
3. Determine which grade range it belongs to.
4. Display the grade.

# Step 5 — Algorithm Design

1. Ask for the score.
2. Check whether it is between 0 and 1.
3. If invalid:

   * Display an error.
4. Otherwise:

   * Compare the score against each grade boundary.
   * Output the matching grade.

# Step 8 — Python

Your solution is correct. The only unnecessary part is:

```python
elif score >= 0.9 or score >= 1:
```

Since you've already validated that the score is at most `1.0`, this is enough:

```python
# Input
score = float(input("Enter Score: "))

# Validation
if score < 0.0 or score > 1.0:
    print("Error: Score is out of range")

# Classification
elif score >= 0.9:
    print("A")

elif score >= 0.8:
    print("B")

elif score >= 0.7:
    print("C")

elif score >= 0.6:
    print("D")

else:
    print("F")
```

## One important observation

You've now learned three fundamental decision patterns:

| Pattern            | Question it answers                   |
| ------------------ | ------------------------------------- |
| **Validation**     | "Is this input acceptable?"           |
| **Selection**      | "Which action should I take?"         |
| **Classification** | "Which category does this belong to?" |

These three patterns will appear repeatedly throughout your programming journey. Recognizing which question a problem is asking is often the fastest way to identify the underlying pattern.
