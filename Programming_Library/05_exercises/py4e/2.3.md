# Step 1 — Problem Analysis

## Purpose

Calculate gross pay from hours worked and hourly rate.

## Inputs

* Hours worked (string from user)
* Rate per hour (string from user)

Example:

```
Hours: 35
Rate: 2.75
```

## Outputs

Gross pay:

```
Pay: 96.25
```

## Constraints

* User enters numbers.
* Convert input strings into floating-point numbers.
* No error checking required.

## Sub-Problems

1. Get hours from user.
2. Get hourly rate from user.
3. Convert text input into numbers.
4. Multiply hours × rate.
5. Display result.
