### Purpose

Prompt the user for their name and greet them.

### Inputs

* User's name (string)

### Outputs

* Greeting message

### Constraints

* Accept any text entered by the user.
* Display the entered name.

### Sub-Problems

1. Ask the user for input.
2. Store the input.
3. Display a greeting using the stored value.
